## 6. 自我研究結論 (我從數字挖出的, 跟用戶直覺不一樣)

### 6.1 用戶 hypothesis 「兩者都極端負 → 下跌」 → **無法驗證, 樣本太少**

- BOTH_NEG (cp_z < -1.5 AND fut_z < -1.5): **n=4** in 6 年, 完全不夠統計
- 6 年大盤 T+20 baseline 平均 +1.71% (= bull market), 「兩者都極端負」 出現得極稀少
- 反推: 法人 OP 跟 期貨等效大台 OI 的 z-score 高度相關, 「兩個都極端負同時發生」 是
  rare event, 不能 build strategy on this

### 6.2 真正強的 signal: **法人意見分歧 (DIVERGE)** → 後續下跌

兩個顯著 finding:

| Pattern | n | 統計顯著 forward return |
|---|---:|---|
| **cp 空 + fut 多** (cp_z<-1, fut_z>+1) | 18 | T+5 **−4.15%** (vs baseline +0.44%, ⬇⬇) |
| **cp 多 + fut 空** (cp_z>+1, fut_z<-1) | 16 | T+20 **−3.06%** (vs baseline +1.71%, ⬇⬇) |

→ **「法人在 OP 跟期貨上意見分歧」是預測下跌的真正信號**, 不是用戶以為的「都極端負」

直覺解讀: 當 OP 跟 期貨 法人方向相反, 通常 是 hedge / 對沖部位拉扯, 反映市場不確定性 →
後續波動加大, 平均偏負

### 6.3 反直覺: **fut 5 日持續極端負 → 後續反而漲** (contrarian)

| Pattern | n | T+10 forward return |
|---|---:|---|
| fut5d 極端負 (5 日平均 z<-1.5) | 51 | **+2.31%** (vs baseline +0.87%, ⬆) |

→ 「期貨法人連續 5 日持續加碼空頭」 不是「下跌前兆」, 反而是**短期底部反彈信號**
→ 屬於 **contrarian indicator** — 過度看空 → 反彈

這正好打破用戶 「持續負部位 → 持續下跌」 的直覺. 現實是: 法人持續看空時, 大盤反而傾向反彈.

### 6.4 fut 看多較強 → 顯著漲 (= momentum 確認)

| Pattern | n | T+20 |
|---|---:|---|
| 同向看多 + fut 較極端 (|fut_z|>|cp_z|, both > 0) | 162 | **+3.01%** (vs +1.71%, ⬆) |

→ 「期貨法人主導看多」 是顯著上漲信號. 跟 6.3 反過來: fut 主導**看多強** = 漲, fut **持續極端空** = 反彈.

### 6.5 小結 (給用戶的 actionable take-aways)

1. **不要只看 cp 或 fut 一個變數** — 它們單獨對 forward return 都 r ≈ 0
2. **看「兩者一致 vs 分歧」** — 分歧 (尤其 cp 空 fut 多) 是強烈下跌信號 (T+5 −4.15%)
3. **「持續看空」 不是下跌前兆** — fut 5 日連續極端負 反而對應 +2.3% T+10 反彈
4. **正常時期 (NEUTRAL) 後續報酬比 baseline 還低** — n=237 都顯示 ⬇, 顯示信號集中在
   極端時期, 中性時期 「無噪音」 但也 「沒 alpha」
5. 樣本限制: 「兩者都極端負」 在 6 年只出現 4 次, 樣本太少不能 build strategy.
   下次重大空頭爆發再說

### 6.6 Caveat / 統計限制

- 6 年資料含 2020-2026 大牛市, 「下跌信號」 可能在純空頭年份效力不同
- z-score 用全段 mean/std 計算, 這是 lookahead bias (應該用 rolling std 才嚴格)
- DIVERGE pattern n=16~18 雖然 |t|>2, 但小樣本顯著性容易誤判 — 看實際 trade 不能完全
  仰賴
- 「分歧→下跌」 的因果機制不明, 可能是 (a) 法人 hedge 行為的副作用 (b) 法人本身
  predictive (c) reverse causation (= 大盤先跌, 然後法人 hedge 部位調整). 純 statistical
  研究無法區分
